# 푸스 카페
## Stemed Glass / Float
- 그레나딘 시럽 1/3
- 잔의 2/3에 바 스푼을 대고 그린 크림 드 민트 1/3
- 잔의 윗부분에 바 스푼을 대고 브랜디 1/3

# 맨해튼
## Cocktail Glass / Stir
- 칵테일 글라스에 얼음을 넣는다
- 버번 위스키 1.5oz(45ml)
- 스위트 베르무트 0.75oz(23ml)
- 앙고스투라 비터스 1 dash
- 바 스푼으로 6회 이상 휘저어 준다
- 스트레이너를 끼우고 얼음을 걸러 컵에 따른다
- 체리로 가니쉬

# 드라이 마티니

# 올드 패션드
## Old-fashioned Glass / Build
- 올드 패션드 글라스에 설탕 1티스푼
- 앙고스투라 비터스 1 dash
- 탄산수 1/2oz (15ml)
- 바 스푼으로 설탕을 저어 녹인다
- 각얼음으로 글라스를 채운다
- 버번 위스키 1.5oz(45ml) 넣고 바 스푼으로 젓는다
- 오렌지 슬라이스, 체리로 가니쉬

# 브랜디 알렉산더

- 칵테일 글라스에 얼음을 넣는다

- 넛맥 파우더로 가니쉬

# 싱가포르 슬링
